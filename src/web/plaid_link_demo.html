Instructions:

  1. Run python link_token_create.py to get a link token
  2. Paste the link token in the input field above
  3. Click "Connect Bank Account" to start Plaid Link
  4. Follow the prompts to connect a bank account (use Sandbox credentials in development)
  5. Copy the public token that appears below
  6. Run python exchange_token.py and paste the public token to get your access token
